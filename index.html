¡Hola, amable bienvenida!

Aquí en este sitio usted encontrará información que le permita conocer sobre los temas de; uso de GIT y GITHUB, estructura básica de html5 y estilos internos y externos de selectores CSS. Explora en el menú de navegación.

Objetivo general del OVI:

Profundizar en el uso de los GIT, GITHUB y CSS3 en gestión de proyectos de diseños web, HTML5 y contenidos en formato web.

Objetivos específicos:

a). Presentar contenidos sobre uso de GIT y GITHUB. b). Conocer estructura básica de html5. c). Mostrar estilos internos y externos selectores CSS.

Referencia: Burgos, W. (s.f.). SOUNDCLOUD. Obtenido de https://soundcloud.com/wendy-burgos-721779057/que-es-git-github-html5-y-css3

Objetivos del curso:

Introducción a HTML5, Estructura básica de HTML5, Aplicación de elementos de HTML5 a un proyecto de diseño Web, Introducción a CSS3, Estructura básica de CSS3, Aplicación de elementos de CSS3 a un proyecto de diseño Web.

Referencia: MACÍAS, R. (s.f.). tooltyp. Obtenido de https://www.tooltyp.com/beneficios-del-html-y-el-css-para-nuestra-marca
